# Student Management System

A simple console-based Java application to manage student records — built to demonstrate core Java and OOP concepts.

## Features
- Add a student
- View all students
- Update student details
- Delete a student
- Search student by ID

## Concepts Demonstrated
- Object-Oriented Programming (Encapsulation)
- Collections (ArrayList)
- Loops & Switch-case
- Exception handling for invalid input

## How to Run
javac *.java
java StudentManagementSystem

## Project Structure
- Student.java – Model class with private fields and getters/setters
- StudentManagementSystem.java – Main class with menu-driven CRUD logic
